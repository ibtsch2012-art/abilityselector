# 🦸‍♂️ Superhero Ability Selector

A simple, interactive command-line interface (CLI) program written in Python that allows users to choose a superhero ability and see their upgraded character stats.

This is one of my very first Python projects as I begin my programming journey! 🚀

---

## 🛠️ Features
* **Interactive UI:** Prompts the user with a list of awesome abilities.
* **Smart Input:** Automatically handles different text casings (e.g., "fly", "FLY", and "Fly" all work!).
* **Instant Feedback:** Shows the selected ability and the corresponding stat boosts (like Acrobatics, Speed, Endurance, or Charisma).
